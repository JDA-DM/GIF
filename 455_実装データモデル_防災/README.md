# 政府相互運用性フレームワーク(GIF) 454 実装データモデル 防災


防災分野のデータモデルを示すガイドブック。このガイドブックは災害時の迅速かつ的確な情報連携を実現することを目的としており、EEI（災害対応基本共有データ属性情報）で定義された情報分類・項目を対象としています。このガイドを参照して情報の提供を行うことで、データの相互運用性を進めることができます。
