# 政府相互運用性フレームワーク(GIF) 454 実装データモデル 教育


教育分野のデータモデルを示すガイドブック。このガイドブックは教育データの相互流通性の確保が目的であり、全国の学校、児童生徒等の属性、学習内容等で共通化できるものを対象としています。このガイドを参照して情報の提供を行うことで、組織横断での教育データの利活用を進めることができます。
