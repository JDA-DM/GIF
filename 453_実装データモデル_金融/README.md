# 政府相互運用性フレームワーク(GIF) 453 実装データモデル 金融

本データモデルは、「IPA DADC 決済テクニカルミーティング」（令和5年5、6月）を踏まえ、
金融分野における相互運用性を確保するための参照データモデルガイドです。
金融の中で様々なサービスを効率的に実現するためのデータ設計基盤として使うことで、
サービス提供者等の関係者は、データ設計や連携の作業・調整を最小化することができます。
