# 政府相互運用性フレームワーク(GIF)  430 コアデータモデル

GIFでは、データモデルを基礎編であるコアデータモデルと、応用編である実装データモデルに大別して整備しています。
コアデータモデルでは、社会の様々な場面で登場する個人や法人、住所や連絡先などの基礎的なデータモデルを定義しています。
